# 💰 DeepMoney: Deep Learning 기반 금융 데이터 예측 시스템

본 문서는 `bgchoi666/DeepMoney` 저장소의 핵심 코드 구조와 각 모듈의 역할을 상세히 설명합니다. 이 프로젝트는 금융 데이터를 효율적으로 관리하고, 딥러닝 모델을 통해 미래 주가나 시장의 흐름을 예측하기 위한 파이프라인으로 구성되어 있습니다.

---

## 1. 프로젝트 개요

`DeepMoney`는 정형 데이터(주가, 지수 등)를 전처리하고, 학습된 모델을 통해 실시간 또는 배치 예측을 수행하는 시스템입니다. 코드의 모듈화(Path -> Reader -> Predict)를 통해 데이터 소스가 바뀌거나 모델이 업데이트되더라도 유연하게 대응할 수 있도록 설계되었습니다.

---

## 2. 주요 파일 구성 및 상세 설명

### 📂 `path.py` (경로 관리 모듈)
* **역할**: 프로젝트 내의 모든 디렉토리 및 파일 경로를 중앙 집중식으로 관리합니다.
* **주요 기능**:
    * 원천 데이터(CSV/Excel)가 저장된 `data/` 경로 설정.
    * 학습된 가중치 파일(`.h5`, `.pth` 등)이 저장되는 `models/` 경로 정의.
    * 예측 결과 및 로그가 저장되는 `output/` 또는 `logs/` 경로 지정.
* **장점**: 하드코딩을 방지하여 협업 환경이나 서버 배포 시 경로 수정 한 번으로 시스템 전체의 파일 접근을 제어할 수 있습니다.

### 📂 `reader.py` (데이터 처리 모듈)
* **역할**: 다양한 소스로부터 데이터를 로드하고 딥러닝 모델이 학습/예측할 수 있는 형태로 전처리합니다.
* **주요 기능**:
    * **Data Ingestion**: Pandas 등을 사용하여 주식 시세나 경제 지표 데이터를 읽어옵니다.
    * **Preprocessing**: 결측치 처리, 데이터 정규화(Scaling), 그리고 기술적 지표(이동평균선, RSI 등)를 생성합니다.
    * **Feature Engineering**: 모델의 입력값이 될 특징(Feature)과 예측 목표(Target)를 설정합니다.
    * **Windowing**: 시계열 분석을 위해 데이터를 특정 타임스텝(Time Step) 단위로 슬라이싱하여 3차원 텐서 형태로 변환합니다.

### 📂 `predict.py` (예측 실행 모듈)
* **역할**: 학습이 완료된 모델을 불러와 실제 예측(Inference)을 수행하는 인터페이스입니다.
* **주요 기능**:
    * **Model Loading**: `path.py`에 정의된 경로에서 최신 가중치 파일을 불러옵니다.
    * **Inference**: `reader.py`를 통해 준비된 최신 데이터를 입력하여 향후 가격이나 등락 확률을 도출합니다.
    * **Post-processing**: 모델의 출력값(0~1 사이의 값 등)을 실제 가격이나 매수/매도 신호로 역변환합니다.
    * **Output**: 예측된 결과를 콘솔에 출력하거나 CSV 파일, DB 등에 저장합니다.

---

## 3. 시스템 워크플로우 (Workflow)

1.  **환경 준비**: `path.py`를 통해 데이터와 모델의 위치를 확인합니다.
2.  **데이터 가공**: `reader.py`가 원시 금융 데이터를 읽어와 모델에 입력 가능한 형태로 정제합니다.
3.  **예측 수행**: `predict.py`가 저장된 모델에 정제된 데이터를 주입하여 예측값을 계산합니다.
4.  **의사 결정**: 도출된 예측값을 바탕으로 투자 전략을 수립하거나 리포트를 생성합니다.

---

## 4. 기술 스택 (Technical Stack)

* **Language**: Python 3.x
* **Libraries**:
    * `Pandas`, `NumPy`: 데이터 분석 및 행렬 연산
    * `TensorFlow` 또는 `PyTorch`: 딥러닝 모델 구축 및 예측
    * `Scikit-learn`: 데이터 전처리 및 스케일링
